      Installed plugins

      Plugins extend the app with custom commands and reactions to events. They run in a sandboxed Web Worker with no network or filesystem access.

        API for plugin authors

        // manifest.json
{
  "id": "my-plugin",
  "name": "My Plugin",
  "version": "1.0.0",
  "entry": "index.js",
  "permissions": ["note:read", "note:write"]
}

// index.js — runs in a Web Worker
mycelium.on("note:saved", (note) => {
  // mutate, notify, transform...
});
mycelium.command("uppercase", (ctx) => {
  ctx.note.body = ctx.note.body.toUpperCase();
  ctx.save();
});
